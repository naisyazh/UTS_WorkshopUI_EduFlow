# Folder Screenshots

Folder ini untuk menyimpan screenshot semua halaman frontend platform EduFlow.

## Cara Mengambil Screenshot

1. **Start Laragon** - Pastikan Apache + MySQL running
2. **Buka halaman di browser:** `http://localhost/UTS_WorkshopUI_EduFlow/nama-halaman.html`
3. **Screenshot:**
   - Tekan `Windows + Shift + S` (Snipping Tool)
   - Atau tekan `F11` untuk fullscreen → `Windows + Shift + S`
4. **Save dengan nama sesuai tabel di bawah**

## Tips

✅ Screenshot saat halaman fully loaded  
✅ Gunakan resolusi minimal 1366x768  
✅ Crop rapi tanpa browser toolbar  
✅ Format: PNG atau JPG  

❌ Jangan screenshot saat ada error  
❌ Jangan screenshot blur atau low res  

## Setelah Screenshot

Setelah semua screenshot selesai:
1. Pastikan semua 20 file ada di folder ini
2. Buka file `LAPORAN-FRONTEND-FINAL.md`
3. Insert screenshot ke tempat yang sesuai
4. Convert ke Word jika perlu untuk submission
