## 🚀 Cara Push ke GitHub (Pilih Salah Satu)

### Option A: Pakai GitHub Desktop (RECOMMENDED ⭐)
1. Download & install GitHub Desktop: https://desktop.github.com/
2. Buka GitHub Desktop
3. File → Add Local Repository
4. Pilih folder: `C:\laragon\www\UTS_WorkshopUI_EduFlow`
5. Login ke GitHub (akun naisyazh)
6. Klik tombol **"Push origin"** (tombol biru di atas)
7. Done! ✅

### Option B: Pakai VS Code
1. Buka VS Code
2. Klik icon "Source Control" di sidebar (ikon cabang)
3. Akan ada 2 commits ready to push
4. Klik "Sync Changes" atau "Push"
5. Done! ✅

### Option C: Pakai Command Line
1. Buat token baru di: https://github.com/settings/tokens
2. Generate token dengan scope `repo`
3. Copy token
4. Jalankan:
```bash
git remote set-url origin https://naisyazh:TOKEN_KAMU@github.com/naisyazh/UTS_WorkshopUI_EduFlow.git
git push origin main
```
